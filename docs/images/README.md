# Screenshots

This folder contains screenshots for the main README.

## Required Screenshots

- **screenshot.png** - Main chat interface showing the AI assistant in action with:
  - Welcome message with examples
  - Example conversation tracking sleep data
  - Markdown-formatted responses with tables
  - Historical query example

## How to Add Screenshots

1. Take a screenshot of the application at https://app-uag4hinooa2z6.azurewebsites.net/
2. Save it as `screenshot.png` in this folder
3. Ensure the screenshot shows:
   - The chat interface
   - Both user and assistant messages
   - Markdown rendering (tables, bold text, bullet points)
   - Example of both tracking current data and querying historical data

## Recommended Size

- Width: 1200-1600px
- Format: PNG with transparency or white background
- Quality: High resolution for GitHub display
